## Installation

1. Clone the repository:

```bash
git clone https://github.com/SHUHAIB-T/todo-app-react.git
cd todo-app-react
```

2. Install dependencies:

```bash
npm install
```

3. Usage:

```bash
npm run dev
```

Open your browser and visit http://localhost:5173 to use the Todo App.
